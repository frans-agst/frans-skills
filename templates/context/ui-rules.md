## Font

Always import [YOUR FONT — e.g. Inter] via `next/font/google` in the root layout.


## Icons

Icons come from **[itshover](https://www.itshover.com/icons)** — animated icons distributed as a shadcn registry (animations built with [Motion](https://motion.dev)). Don't hand-roll SVG icons or pull in a second icon library when an itshover icon fits.

Install each icon you need via the shadcn CLI (use the project's package manager — npm/pnpm/yarn/bun):

```bash
npx shadcn@latest add https://itshover.com/r/[icon-name]-icon.json
```

This drops a React component into the project; import and use it like any component. Each icon has its own page at `itshover.com/icons/[icon-name]` with the exact install command and copyable code.

- Requires a **shadcn-configured** project (`components.json` present).
- `/design-system` records which itshover icons a feature uses in its per-feature section of `context/design-system.md`.
- **License:** not stated on the site — confirm on the [itshover GitHub repo](https://github.com/itshover/itshover) before shipping.
